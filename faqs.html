PadCalc FAQs

1. What is PadCalc?
PadCalc is a calculator embedded within a notepad. It allows you to write operations, and the pad will calculate the results for you. You can chain operations and save the results in variables for use in other calculations.
2. What operations are supported?
PadCalc supports addition, subtraction, multiplication, division, modulus (8%3=2), and power (2**4=16) operations. Also, it understands parenthesis and the order of operations: 12/(6+6)=1.
3. How can I create variables?
You can add variables in the left column using the 'Add Variable' button, or use the results of your operations with the '->' symbol.
For example: 2+3=5->var
4. What characters are allowed in variable names?
Variable names must start with letters and can contain letters, numbers, or underscores. For example: var_1.
5. Can I save and restore my notes?
Yes, there is an 'Export' button that allows you to download a JSON file with the data.
Then, you can use the 'Import' button to upload this file or another containing PadCalc data.
6. How can I share/send my calculations with someone else?
You can send the exported file by email, slack, whatsapp...
7. Do you store or sell my data to others?
No, this application runs only in your browser. There isn't any database or application server behind it.
